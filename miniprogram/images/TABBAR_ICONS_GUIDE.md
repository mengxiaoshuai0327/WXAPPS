# TabBar 图标制作指南

## 图标样式要求

根据设计图，TabBar 图标需要：

### 1. 首页图标
- **未选中**：灰色轮廓房子图标
- **选中**：蓝色/深色实心房子图标

### 2. 课程表图标
- **未选中**：灰色轮廓日历图标（带勾选标记）
- **选中**：蓝色/深色实心日历图标

### 3. 排行榜图标
- **未选中**：灰色轮廓奖杯图标
- **选中**：蓝色/深色实心奖杯图标

### 4. 我的图标
- **未选中**：灰色轮廓人物图标
- **选中**：蓝色/深色实心人物图标

## 快速获取图标

### 推荐方案：IconFont（阿里巴巴图标库）

1. **访问**：https://www.iconfont.cn/

2. **搜索并下载图标**：

   **首页图标**：
   - 搜索：`home` 或 `首页`
   - 选择：实心房子图标（选中状态）
   - 选择：轮廓房子图标（未选中状态）
   - 下载时选择：PNG，81px
   - 颜色：选中用 #1a1a1a，未选中用 #999999

   **课程表图标**：
   - 搜索：`calendar` 或 `日历` 或 `schedule`
   - 选择：带勾选的日历图标
   - 下载时选择：PNG，81px

   **排行榜图标**：
   - 搜索：`trophy` 或 `奖杯` 或 `ranking`
   - 选择：奖杯图标
   - 下载时选择：PNG，81px

   **我的图标**：
   - 搜索：`user` 或 `用户` 或 `profile`
   - 选择：人物图标
   - 下载时选择：PNG，81px

3. **调整颜色**：
   - 使用图片编辑工具（如 Photoshop、在线工具）
   - 选中状态：改为 #1a1a1a（深色）
   - 未选中状态：改为 #999999（灰色）

4. **保存文件**：
   ```
   images/
   ├── home.png
   ├── home-active.png
   ├── schedule.png
   ├── schedule-active.png
   ├── ranking.png
   ├── ranking-active.png
   ├── profile.png
   └── profile-active.png
   ```

## 在线工具推荐

1. **IconFont**：https://www.iconfont.cn/（最推荐）
2. **Remix Icon**：https://remixicon.com/
3. **Heroicons**：https://heroicons.com/
4. **Canva**：https://www.canva.com/（在线编辑）

## 临时方案

如果暂时没有图标文件，可以：
1. 使用文字模式（当前配置）
2. 使用简单的 Unicode 字符作为占位
3. 使用在线图标生成器快速生成

## 注意事项

- 图标尺寸：81px × 81px（推荐）
- 格式：PNG（支持透明背景）
- 文件大小：建议 < 50KB
- 确保图标清晰，在小屏幕上也能看清
- 选中和未选中图标要保持相同的视觉大小
