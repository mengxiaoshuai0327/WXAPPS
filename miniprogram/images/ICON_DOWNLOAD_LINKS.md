# TabBar 图标下载链接

## 推荐下载方式

### 方式1：IconFont（阿里巴巴图标库）- 最推荐

**网址**：https://www.iconfont.cn/

#### 首页图标
1. 搜索：`home` 或 `首页`
2. 推荐图标ID（示例）：
   - 实心：搜索 "home" 选择实心样式
   - 轮廓：搜索 "home outline" 选择轮廓样式
3. 下载：PNG，81px

#### 课程表图标
1. 搜索：`calendar` 或 `schedule` 或 `日历`
2. 推荐：选择带勾选标记的日历图标
3. 下载：PNG，81px

#### 排行榜图标
1. 搜索：`trophy` 或 `奖杯` 或 `ranking`
2. 推荐：选择奖杯图标
3. 下载：PNG，81px

#### 我的图标
1. 搜索：`user` 或 `用户` 或 `profile`
2. 推荐：选择人物图标
3. 下载：PNG，81px

## 快速操作步骤

1. **访问 IconFont**：https://www.iconfont.cn/
2. **搜索图标**：依次搜索上述关键词
3. **选择样式**：
   - 未选中：选择轮廓/线性样式
   - 选中：选择实心/填充样式
4. **下载设置**：
   - 格式：PNG
   - 尺寸：81px × 81px
5. **调整颜色**（使用在线工具如 https://www.iloveimg.com/resize-image）：
   - 未选中图标：改为 #999999
   - 选中图标：改为 #1a1a1a
6. **重命名并保存**到 `miniprogram/images/` 目录

## 文件命名

下载后重命名为：
- `home.png` / `home-active.png`
- `schedule.png` / `schedule-active.png`
- `ranking.png` / `ranking-active.png`
- `profile.png` / `profile-active.png`

## 注意事项

- 图标尺寸必须是 81px × 81px
- 格式必须是 PNG（支持透明背景）
- 确保图标清晰，在小屏幕上也能看清
- 选中和未选中图标要保持相同的视觉大小
