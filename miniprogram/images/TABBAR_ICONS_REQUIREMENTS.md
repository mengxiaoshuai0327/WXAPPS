# TabBar 图标要求（根据设计图）

## 图标样式规范

根据设计图，TabBar 图标需要以下样式：

### 1. 首页（Home）
- **未选中**：灰色轮廓房子图标
- **选中**：蓝色/深色实心房子图标
- **颜色**：未选中 #999999，选中 #1a1a1a

### 2. 课程表（Schedule）
- **未选中**：灰色轮廓日历图标（带勾选标记）
- **选中**：蓝色/深色实心日历图标
- **颜色**：未选中 #999999，选中 #1a1a1a

### 3. 排行榜（Ranking）
- **未选中**：灰色轮廓奖杯图标
- **选中**：蓝色/深色实心奖杯图标
- **颜色**：未选中 #999999，选中 #1a1a1a

### 4. 我的（Profile）
- **未选中**：灰色轮廓人物图标
- **选中**：蓝色/深色实心人物图标
- **颜色**：未选中 #999999，选中 #1a1a1a

## 快速获取图标

### 推荐：IconFont（阿里巴巴图标库）

1. **访问**：https://www.iconfont.cn/

2. **搜索图标**：
   - 首页：搜索 `home` 或 `首页`，选择实心和轮廓两种样式
   - 课程表：搜索 `calendar` 或 `schedule`，选择带勾选的日历图标
   - 排行榜：搜索 `trophy` 或 `奖杯`
   - 我的：搜索 `user` 或 `用户`

3. **下载设置**：
   - 格式：PNG
   - 尺寸：81px × 81px
   - 颜色：下载后使用图片编辑工具调整
     - 未选中：改为 #999999（灰色）
     - 选中：改为 #1a1a1a（深色）

4. **文件命名**：
   ```
   home.png / home-active.png
   schedule.png / schedule-active.png
   ranking.png / ranking-active.png
   profile.png / profile-active.png
   ```

5. **放入目录**：
   将所有图标文件放入 `miniprogram/images/` 目录

## 图标已配置

`app.json` 中已配置图标路径，添加图标文件后即可显示。

## 临时方案

如果暂时没有图标文件，TabBar 会只显示文字（当前状态），等图标准备好后会自动显示。
