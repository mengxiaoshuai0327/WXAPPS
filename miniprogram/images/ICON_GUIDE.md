# 小程序 TabBar 图标获取指南

## 图标要求
- **尺寸**: 81px × 81px（推荐）
- **格式**: PNG（支持透明背景）
- **数量**: 每个 tab 需要 2 个图标（未选中 + 选中状态）
- **风格**: 简洁、商务风格

## 推荐的免费图标资源

### 1. 阿里巴巴 IconFont（最推荐）
**网址**: https://www.iconfont.cn/

**使用方法**:
1. 搜索关键词：首页、课程、排行榜、我的
2. 选择简洁的线性或面性图标
3. 下载 PNG 格式，尺寸选择 81px
4. 可以调整颜色（未选中：#999999，选中：#1a1a1a）

**推荐图标搜索词**:
- 首页：`home`、`首页`、`house`
- 课程表：`schedule`、`课程`、`calendar`
- 排行榜：`ranking`、`排行榜`、`trophy`
- 我的：`user`、`我的`、`profile`

### 2. Iconify（国际化图标库）
**网址**: https://iconify.design/

**特点**:
- 包含多个图标库（Material Design、Font Awesome 等）
- 可以导出为 PNG/SVG
- 支持自定义颜色和尺寸

### 3. Remix Icon（简洁线性图标）
**网址**: https://remixicon.com/

**特点**:
- 开源免费
- 线性风格，简洁商务
- 可以直接下载 SVG，转换为 PNG

### 4. 即时设计（国内）
**网址**: https://js.design/

**特点**:
- 提供小程序图标合集
- 包含 WeUI 图标库
- 可以直接下载使用

## 快速获取方案

### 方案一：使用 IconFont（推荐）

1. 访问 https://www.iconfont.cn/
2. 注册/登录账号
3. 搜索并选择以下图标：
   - **首页**: 搜索 "home" 或 "首页"
   - **课程表**: 搜索 "calendar" 或 "课程"
   - **排行榜**: 搜索 "trophy" 或 "排行榜"
   - **我的**: 搜索 "user" 或 "我的"
4. 添加到购物车，批量下载
5. 下载时选择 PNG 格式，尺寸 81px
6. 使用图片编辑工具调整颜色：
   - 未选中状态：灰色 (#999999)
   - 选中状态：深色 (#1a1a1a)

### 方案二：使用在线图标生成器

1. **Canva**: https://www.canva.com/
   - 搜索 "小程序图标" 模板
   - 自定义颜色和文字
   - 导出为 PNG

2. **稿定设计**: https://www.gaoding.com/
   - 搜索 "小程序图标"
   - 在线编辑
   - 一键下载

### 方案三：使用 AI 生成图标

可以使用 AI 工具（如 Midjourney、DALL-E）生成图标：
- 提示词示例：`简洁商务风格的小程序图标，首页图标，线性风格，81x81像素，白色背景`

## 图标命名规范

下载后，请按以下方式命名并放入 `images/` 目录：

```
images/
├── home.png              # 首页未选中
├── home-active.png       # 首页选中
├── schedule.png          # 课程表未选中
├── schedule-active.png   # 课程表选中
├── ranking.png           # 排行榜未选中
├── ranking-active.png    # 排行榜选中
├── profile.png           # 我的未选中
└── profile-active.png    # 我的选中
```

## 图标颜色建议

根据你的小程序主题色（#1a1a1a），建议：

- **未选中状态**: #999999（灰色）
- **选中状态**: #1a1a1a（深色/黑色）

## 临时解决方案

如果暂时没有图标，可以：

1. **使用文字图标**：在 tabBar 中只显示文字，不显示图标
2. **使用占位图标**：创建简单的几何图形作为占位
3. **使用系统图标**：使用微信小程序内置的图标字体

## 图标设计建议

1. **风格统一**：所有图标使用相同的设计风格（线性或面性）
2. **简洁明了**：图标要清晰易懂，符合用户认知
3. **对比度**：确保在白色背景上清晰可见
4. **尺寸一致**：所有图标保持相同的视觉大小

## 快速开始

**最快的方法**：
1. 访问 https://www.iconfont.cn/
2. 搜索 "home"、"calendar"、"trophy"、"user"
3. 选择简洁的线性图标
4. 下载 PNG 格式，81px
5. 使用在线工具（如 https://www.iloveimg.com/resize-image）调整颜色
6. 重命名并放入 `images/` 目录
7. 在 `app.json` 中恢复图标路径配置

## 注意事项

- 确保图标文件大小适中（建议 < 50KB）
- 使用透明背景的 PNG 格式
- 检查图标的授权协议，确保可以商用
- 保持图标风格与小程序整体设计一致
