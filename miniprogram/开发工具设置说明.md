# 微信开发者工具设置说明

## 设置不校验域名（重要！）

为了在开发环境中使用本地 IP 地址访问后端 API，需要在微信开发者工具中关闭域名校验。

### 操作步骤：

1. **打开微信开发者工具**
   - 打开你的小程序项目

2. **进入设置页面**
   - 点击右上角的 **"详情"** 按钮
   - 或使用快捷键：`Cmd + ,` (Mac) / `Ctrl + ,` (Windows)

3. **本地设置**
   - 在左侧菜单中找到 **"本地设置"** 选项卡
   - 勾选以下选项：
     - ✅ **不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书**
     - ✅ **不校验安全域名**

4. **保存并重新编译**
   - 关闭设置窗口
   - 点击工具栏的 **"编译"** 按钮
   - 或使用快捷键：`Cmd + R` (Mac) / `Ctrl + R` (Windows)

## 验证设置

设置完成后，小程序应该能够正常访问后端 API：
- 课程表页面可以加载课程数据
- 主题筛选功能正常
- 课程详情可以查看

## 常见问题

### 问题1：仍然无法访问 API
- 检查后端服务是否运行：`curl http://localhost:3000/health`
- 检查 API 地址是否正确：查看 `miniprogram/app.js` 中的 `apiBaseUrl`
- 查看控制台错误信息

### 问题2：网络请求失败
- 确保手机和电脑在同一 WiFi 网络
- 如果使用手机预览，需要将 IP 地址改为电脑的局域网 IP
- 检查防火墙设置

### 问题3：API 地址配置
当前配置的 API 地址：`http://192.168.124.6:3000/api`

如果需要修改，编辑 `miniprogram/app.js`：
```javascript
apiBaseUrl: 'http://你的IP地址:3000/api'
```

## 生产环境配置

**注意**：这些设置仅用于开发环境。发布到生产环境时：
1. 需要配置合法的 HTTPS 域名
2. 在微信公众平台配置服务器域名
3. 将 API 地址改为生产环境的 HTTPS 地址
