# 示例数据插入说明

## 功能

此脚本会在数据库中插入示例课程数据，包括：
- 3个课程模块
- 6个课程主题
- 3位授课人
- 6门课程
- 30个排课（未来30天内）

## 使用方法

### 前置条件

1. **MySQL 服务已启动**
   ```bash
   # macOS
   brew services start mysql
   
   # 检查服务状态
   brew services list | grep mysql
   ```

2. **数据库已初始化**
   ```bash
   cd backend
   npm run init-db
   ```

3. **数据库配置正确**
   确保 `backend/.env` 文件中的数据库配置正确：
   ```env
   DB_HOST=localhost
   DB_PORT=3306
   DB_USER=root
   DB_PASSWORD=your_password
   DB_NAME=xiaocx_db
   ```

### 执行插入

```bash
cd backend
npm run insert-sample
```

### 验证数据

插入成功后，您可以在小程序中查看：
- 课程表页面：显示所有排课
- 课程详情：点击"详情"查看课程信息
- 筛选功能：按主题和日期筛选课程

## 注意事项

- 如果数据库中已有数据，脚本会继续插入新数据
- 排课日期设置为未来30天内
- 每门课程会创建5个排课
- 排课时间段随机分配（上午/下午/全天）

## 清除示例数据

如果需要清除示例数据，可以手动删除或重新初始化数据库：

```bash
# 重新初始化数据库（会清空所有数据）
npm run init-db

# 然后重新插入示例数据
npm run insert-sample
```
