# MXS 授课人示例数据

## 功能说明

这个脚本专门为授课人 **MXS** 创建完整的示例数据，包括：
- 课程信息（4门课程）
- 已完成排课（历史课程）
- 未来排课（即将开始的课程）
- 评价数据（学员评价和评分）

## 使用方法

```bash
cd backend
node scripts/insert-mxs-instructor-samples.js
```

## 创建的数据详情

### 1. 授课人信息
- **用户名**: MXS
- **角色**: instructor（授课人）
- **用户ID**: 4（如果已存在）或新创建的ID

### 2. 课程列表（4门）

1. **高效项目管理实战** (PM001)
   - 副标题：掌握项目管理核心技能，提升项目成功率
   - 课程介绍：深入讲解项目管理的核心概念和实践技巧

2. **团队协作与沟通** (TC001)
   - 副标题：打造高效团队，提升协作效率
   - 课程介绍：学习团队协作的关键要素，掌握有效沟通技巧

3. **数据分析与决策** (DA001)
   - 副标题：用数据驱动业务决策
   - 课程介绍：学习数据分析方法，掌握数据可视化技巧

4. **领导力提升训练** (LEAD001)
   - 副标题：培养卓越领导力，成为优秀管理者
   - 课程介绍：系统学习领导力理论和实践

### 3. 排课数据

#### 已完成排课
- 每门课程：**5期**已完成排课
- 排课日期：过去35天内（每7天一期）
- 状态：`completed`
- 问卷状态：已触发

#### 未来排课
- 每门课程：**3期**未来排课
- 排课日期：未来30天内（每10天一期）
- 状态：`scheduled`
- 问卷状态：未触发

### 4. 评价数据

- **评价数量**：每期4-6个评价（随机）
- **总评价数**：约112个评价
- **评分范围**：3.2 - 4.8分
- **评分特点**：
  - 不同课程有不同基础评分（3.8-4.5分）
  - 早期评分较低，后期评分逐渐提升
  - 包含随机波动（±0.25分），模拟真实评价

### 5. 评价内容

每个评价包含：
- Q1-Q4, Q9 评分选项
- 文字反馈建议
- 提交时间（课程后1天）

示例反馈：
- "课程内容很实用，帮助我解决了实际问题。"
- "MXS老师的讲解很清晰，案例丰富，学到了很多。"
- "通过这门课程，我对相关知识有了更深入的理解。"

## 查看方式

### 1. 小程序查看

#### 课程评价
1. 使用MXS账号登录小程序
2. 进入【我的】页面
3. 点击【授课人专区】
4. 点击【课程评价】
5. 查看所有课程的评分统计

#### 未来开课
1. 进入【首页】
2. 点击【课程表】
3. 查看未来30天内的开课安排
4. 可以看到每门课程的报名人数等信息

### 2. API测试

#### 获取课程评价列表
```bash
curl "http://localhost:3000/api/evaluations/instructor/courses?instructor_id=4"
```

#### 获取课程表（包含未来开课）
```bash
curl "http://localhost:3000/api/courses/schedule?user_id=4"
```

#### 获取已完成课程
```bash
curl "http://localhost:3000/api/courses/completed?user_id=4"
```

## 数据特点

1. **完整性**：包含完整的课程生命周期数据
   - 课程创建
   - 历史排课和评价
   - 未来排课安排

2. **真实性**：模拟真实教学场景
   - 评分逐步提升（课程改进）
   - 不同课程有不同的平均分
   - 评价内容多样化

3. **可测试性**：涵盖各种功能点
   - 课程评价查看
   - 未来开课查看
   - 评分统计和趋势

## 重新运行脚本

如果数据被修改或需要重置，可以再次运行脚本：
- 脚本会检查现有数据，避免重复创建
- 如果需要完全重置，可以先清空相关数据

## 注意事项

1. 确保数据库连接正常
2. 确保存在至少一个课程主题
3. 如果MXS用户不存在，脚本会自动创建
4. 如果MXS用户不是授课人角色，脚本会自动切换
