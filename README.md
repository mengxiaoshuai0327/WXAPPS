# CFO高手私教小班课小程序

## 项目简介
一个完整的教育培训类小程序，包含课程预约、课券管理、评价系统、邀请功能等完整业务功能。

## 项目结构
```
Xiaocx/
├── miniprogram/          # 小程序前端
├── backend/              # 后端API服务
├── admin/                # 管理后台
└── docs/                 # 文档
```

## 技术栈
- **小程序前端**: 微信小程序原生框架
- **后端**: Node.js + Express + MySQL
- **管理后台**: Vue.js + Element UI

## 快速开始

### 1. 后端服务
```bash
cd backend
npm install
npm start
```

### 2. 小程序
使用微信开发者工具打开 `miniprogram` 目录

### 3. 管理后台
```bash
cd admin
npm install
npm run serve
```

## 功能模块
- 用户系统（游客/会员/授课人）
- 课券管理（购买/使用/赠予/开发票）
- 课程管理（课程表/预订/取消）
- 课后评价系统
- 邀请与推荐
- 折扣券系统
- 排行榜
- 管理后台
