## 后端API

### 营销方案管理API
- `GET /api/admin/marketing/list` - 获取所有营销方案
- `POST /api/admin/marketing/create` - 创建营销方案
- `PUT /api/admin/marketing/:id` - 更新营销方案
- `DELETE /api/admin/marketing/:id` - 删除营销方案
- `GET /api/admin/marketing/invitation-stats` - 获取授课人/渠道方邀请统计

### 渠道方管理API
- `GET /api/admin/channels` - 获取渠道方列表
- `POST /api/admin/channels` - 创建渠道方
- `PUT /api/admin/channels/:id` - 更新渠道方

## 使用说明

### 营销方案管理
1. 进入"营销方案管理"页面
2. 点击"创建营销方案"按钮
3. 填写方案信息：
   - 方案名称
   - 折扣比例（如0.10表示10%折扣）
   - 适用角色（可多选：授课人、渠道方）
   - 方案描述（可选）
   - 状态（激活/未激活）
4. 保存后，系统会在用户通过授课人/渠道方邀请注册时自动应用该折扣

### 邀请统计
1. 进入"邀请管理"页面
2. 切换到"授课人/渠道方邀请统计"Tab
3. 可以查看所有授课人和渠道方的邀请数据
4. 可以按角色或用户ID筛选

### 创建渠道方
1. 进入"用户管理" → "渠道方列表"
2. 点击"创建渠道方"按钮
3. 填写渠道方信息：
   - 基本信息：昵称、姓名、手机号、密码
   - 渠道信息：渠道名称、渠道编码、联系人、联系电话、描述
   - 头像（可选）
4. 系统会自动生成渠道方ID（格式：CH + 8位数字）

## 注意事项

1. **折扣比例格式**：营销方案中的折扣比例是小数形式（0.1 = 10%），不是百分比
2. **营销方案优先级**：如果有多个激活的方案，系统会取最新创建的方案
3. **渠道方ID**：创建渠道方时，系统会自动生成唯一的渠道方ID，可作为邀请码使用
4. **邀请统计**：统计数据实时反映授课人/渠道方的邀请效果，用于营销分析
